## 注意事项

只要你的Mac能接收到iPhone短信就可以使用   [如何将消息从iPhone同步到Mac电脑，分步指南](https://www.duotin.com/ios/68361.html)

## 使用

**直接在utools框内输入 下列任何关键词都可以 触发**   

>  "cmds": ["1", "2", "11", "12", "code", "验证码", "向前", "rational_stars"]

推荐使用1 因为比较迅速   ，直接回车验证码就可以直接粘贴验证码了，command+1 也行。
